# How Many Smokes?

Live map of wildfire smoke over Ontario, and what breathing it costs you in
cigarettes.

```sh
python devserver.py 5177   # http://localhost:5177
```

Data: [ECCC GeoMet](https://eccc-msc.github.io/open-data/msc-data/nwp_raqdps-fw/readme_raqdps-fw_en/)
(smoke, PM2.5) and CWFIS (fires). Method:
[Berkeley Earth](https://berkeleyearth.org/air-pollution-and-cigarette-equivalence/).

A risk analogy, not chemistry. Not medical advice.
